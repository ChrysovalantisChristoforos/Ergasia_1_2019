Our Future Project

Μια ιδέα που έχουμε σκεφτεί και θα θέλαμε να την υλοποιήσουμε στο μέλλον είναι μια
εφαρμογή-ιστοσελίδα για να βοηθήσουμε φιλανθρωπικούς σκοπούς που δεν είναι τόσο γνωστοί
στον κόσμο. Θα περιλαμβάνει ένα eshop όπου ο κάθε χρήστης θα μπορεί να ανεβάσει μια φωτογραφία
από ενα πράγμα ή ρούχα που δεν χρειάζεται πια. Με σκοπό ο κόσμος που θα θέλει να βοηθήσει
αυτές τις οργανώσεις να αγοράσει αυτά που θα υπάρχουν στην πλατφόρμα, με αρκέτα λογικές τιμές,
και το 80% των χρημάτων θα πηγαίνει στις οργανώσεις αυτες. Όπου θα εναλλάσονται συνεχώς
το ποιες θα είναι κάθε μήνα. Ώστε να παρέχουμε όσοι περισσότερη βοήθεια μπορούμε.
